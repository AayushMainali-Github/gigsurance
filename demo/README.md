# Demo Launcher

Minimal one-page launcher for the project demos.

## Setup

1. Copy `.env.example` to `.env`
2. Set the two target links:
   - `VITE_DEMO_LINK_ONE` for the finance admin panel
   - `VITE_DEMO_LINK_TWO` for the mock monitor admin panel

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
